AndroidDeviceNames

A fork of jaredrummler/AndroidDeviceNames — a small Android library that converts device model names (like SM-N910W8) into human-readable names (like Samsung Galaxy Note 4). This fork adds automated releases so it will never get abandoned.


View on GitHub →

Why this fork?

🤖 Automated Releases

Releases are automated via CI/CD, so the library stays up-to-date without manual intervention.

🚫 Never Abandoned

Unlike the original, this fork is designed to keep working long-term with no risk of going unmaintained.

📱 27,000+ Devices

Supports over 27,000 Android devices based on Google's maintained device list.

Installation

// Add to your module's build.gradle
implementation 'io.github.antonio2904:android-device-names:2026.06.00'

Usage

// 1. Initialize once in your Application class
DeviceName.init(this);

// 2. Get current device name (UI thread safe)
String name = DeviceName.getDeviceName();

// 3. Get name by codename
DeviceName.getDeviceName("clark", "Unknown"); // → "Moto X Style"

// 4. Async full database lookup
DeviceName.with(context).request((info, error) -> {
  String marketName = info.marketName; // e.g. "Galaxy S8+"
  String model = info.model; // e.g. "SM-G955W"
});
